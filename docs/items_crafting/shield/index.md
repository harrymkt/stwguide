# Shield

a protection object, a Shield.

## Raw Resources Required
* 80 Grass or 8 String
* 8 metal
## Tools Required
* 1 Hammer
* 1 fire for forging

# step1: Grass + Grass
Makes 1 string
Repeat process, shield requires 8 String
Note: this will require 80 Grass, 10 Grass per string
# step 2: forge metal + hammer
Makes 1 bent metal
Repeat process, shield requires 8 Bent Metal
# step 3: Bent Metal + String
makes 1 Tied Metal
Repeat process, shield requires 8 Tied Metal
# step 4: Tied Metal + Tied Metal
Makes 1 Tied Metal Pieces
Repeat process, shield requires 4 Tied Metal Pieces
# step 5: Tied Metal Pieces + Tied Metal Pieces
Makes 1 Large Tied Metal Pieces
Repeat process, shield requires 2 Large Tied Metal Pieces
# step 6: Large Tied Metal Pieces + Large Tied Metal Pieces
Makes 1 **Shield**
