# Spit

## Raw Resources Required
* 4 Wood
## Tools Required
* 1 Knife

# step1: Knife + Wood
Makes 1 Carved Wood
Repeat process, Spit requires 4 Carved Wood
# step2: Knife + Carved Wood
Makes 1 Shaft
Repeat process, Spit requires 4 Shaft
# step3: Shaft + Shaft
Makes 1 Long Shaft
Repeat process, Spit requires 2 Long Shaft
# step4: Long Shaft + Long Shaft
Makes 1 **Spit**
